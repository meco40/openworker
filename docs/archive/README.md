# Archive Index

Stand: 2026-02-21

Dieses Verzeichnis enthaelt historische Dokumente. Sie bleiben aus Nachvollziehbarkeit erhalten,
sind aber keine aktive Source of Truth.

## Struktur

- `docs/archive/legacy-core/` (alte Kernsystem-Doku)
- `docs/archive/analysis/` (Deep-Dives, Vergleiche, Architektur-Analysen)
- `docs/archive/reviews/` (Reviews und Audits aus frueheren Staenden)
- `docs/archive/reports/` (Implementierungsberichte, Changelogs, Mock-Reports)
- `docs/archive/architecture-proposals/` (fruehere Architekturvorschlaege)
- `docs/archive/rooms/` (historische Rooms-Analysen)
- `docs/archive/operations/` (ersetzte Betriebsdokumente)
- `docs/archive/plans/completed/` (abgeschlossene Plaene)
- `docs/archive/plans/superseded/` (ersetzte/ueberholte Plaene)
- `docs/archive/plans/ideas/` (Ideen ohne Umsetzung)

## Ablageregeln

1. Alles mit Fokus auf Vergangenheitsanalyse oder Vergleich kommt nach `analysis/`.
2. Umsetzungsberichte ohne laufenden Betriebswert kommen nach `reports/`.
3. Plan-Dokumente werden nach Status in `plans/completed` oder `plans/superseded` einsortiert.

## Regel

Bei Widerspruechen gilt immer die aktive Doku unter `docs/` ohne `docs/archive/`.
