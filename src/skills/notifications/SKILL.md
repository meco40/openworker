Use this skill to send notifications to the user via various channels.

When a task completes, an important event occurs, or the user requests a reminder, use `notify` to send a push notification or alert.

Keep notification text concise and actionable. Include a title and brief body. Only send notifications for genuinely important events — avoid spam.
