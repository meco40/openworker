Use this skill to interact with GitHub repositories, issues, pull requests, and code.

When a user asks about GitHub repos, wants to create issues, review PRs, or manage repository content, use the github skill functions.

Always ask for the repository owner/name before operating. Prefer read operations before writes. When creating issues or PRs, use clear, descriptive titles and bodies.
