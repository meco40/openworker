Use this skill to fetch the text content of any public URL.

When a user asks you to read a webpage, documentation, article, or API response, use `web_fetch`. Returns cleaned text after stripping HTML. Does not execute JavaScript.

Great for: reading documentation, extracting article content, checking API responses, scraping public data. Not suitable for pages that require login or JavaScript rendering.
