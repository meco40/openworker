Use this skill to read, write, and manage files on the host filesystem.

When a user asks you to read a file, create a file, list a directory, or make changes to the codebase, use the filesystem skill functions (`file_read`, `file_write`, `file_list`, etc.).

Always confirm before overwriting or deleting files. Prefer reading before writing to understand current content.
