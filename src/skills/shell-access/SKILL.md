Use this skill to execute shell commands and scripts on the host system.

When a user asks you to run a command, install software, check system status, or perform any terminal operation, use `shell_exec`.

Always explain what a command does before running it. Prefer safe, reversible commands. For destructive operations (rm, format, etc.), confirm intent first.

On Windows, commands run in PowerShell. On Unix, commands run in bash/sh.
